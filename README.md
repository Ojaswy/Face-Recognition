# Face-Recognition

A Face-Recognition software brought to life by the *OpenCV* Library. 

## Requirements

-> Python 3.5(.py) & 2.7(webcam_detect.py)

-> Numpy

-> Scipy

-> OpenCV installation -> pip3 install opencv-python

## Results
